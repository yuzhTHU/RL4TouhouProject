# RL for Touhou Project

一个试图用 RL 玩东方 Project 的程序

## Setup
1. 安装 tesseract-OCR：参考 https://www.jianshu.com/p/f7cb0b3f337a
2. `pip install -r requirements.txt`

## 其它信息
